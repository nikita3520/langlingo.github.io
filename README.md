# langlingo

# 🚀 О проекте
Langlingo — это приложение для изучения иностранных языков.

Проект создан как учебный прототип приложения Duolingo.

# ✨ Функции
### 1. Уроки. Изучение иностранных языков.
### 2. создание профиля.
### 3. Возможность оформления подписки.
### 4. История. Возможность просмотра пройденных уроков.
### 5. Тренировки. Повторение пройденного материала.
